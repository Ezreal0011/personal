# 更新日志 v1.0.1 (2026-04-25)

## 本次更新内容

### 资源节点卡优化

1. **资源节点卡汇总值显示**
   - 在资源节点卡中间显示输入边的汇总数值
   - 显示格式：`x总数值`（如 x100）
   - 数值颜色使用产出色（绿色）

2. **汇总值计算逻辑**
   - 优先读取**输出端口的 portValue**（在系统节点属性面板中配置）
   - 回退使用边的 value 字段
   - 支持多边数值累加

3. **卡片高度自适应**
   - 高度 = 头部 + 汇总区域 + 端口区域
   - 根据端口数量自动调整

4. **输入/输出端口布局**
   - 左侧：输入端口（红色圆点，竖排）
   - 右侧：输出端口（绿色圆点，竖排）

### 交互优化

1. **边选择时自动打开属性面板**
   - 点击边即可编辑属性，无需右键菜单
   - 快速配置数值

### 已知问题

1. 边的 value 字段可能未被正确保存
   - 建议通过**输出端口配置数值**（portValue）
   - 在系统节点属性面板的输出端口区域配置

## 配置数值步骤

1. 在画布上选择**系统节点**（如"普通章节"）
2. 在右侧属性面板中找到**输出端口**区域
3. 点击端口配置按钮，设置 **portValue**（数值）
4. 从该端口连线到资源节点
5. 资源节点卡中间将自动显示汇总值
