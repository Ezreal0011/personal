# 更新日志 v1.0.2 (2026-04-25)

## 本次更新内容

### 1. 框选与多选功能

- **Ctrl+拖动框选**：按住Ctrl键在画布上拖动矩形选框，选中框内的所有节点
- **Ctrl+点击多选**：按住Ctrl键点击节点，可逐个添加/取消选择
- **批量移动**：选中多个节点后，拖动任意一个节点，所有选中节点同时移动
- **取消选择**：点击空白区域清除所有选择

### 2. 节点高亮逻辑优化

- 点击根级父节点（如"主线地图"、"人物养成"）时，高亮其所有子节点
- 点击普通节点时，高亮一级关联节点，其他节点变暗
- 选中边时，高亮该边的两端节点

### 3. 小地图同步

- 小地图只显示画布上可见的节点（不包括根级父节点）

### 4. 左侧系统树点击

- 点击左侧系统树节点时，右侧弹出属性编辑面板

## 使用说明

### 框选操作

1. **Ctrl+拖动** - 在画布上框选矩形区域
2. **Ctrl+点击** - 逐个添加/取消选择节点
3. **拖动节点** - 批量移动选中的节点
4. **点击空白** - 取消所有选择
